# Dataset Evaluation

Sprint 5 - Hadith Data Integration

---

## Tujuan

Menentukan dataset hadis yang akan menjadi fondasi Project Mizan.

Pemilihan dataset harus mempertimbangkan:

- legalitas penggunaan
- kualitas data
- kelengkapan
- kemudahan pemeliharaan
- skalabilitas jangka panjang

---

Layak dipertimbangkan.
