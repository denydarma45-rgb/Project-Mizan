# Tujuan

Dokumen ini mendefinisikan seluruh sumber data yang digunakan oleh Project Mizan.

Seluruh data yang masuk ke database wajib memiliki:

- sumber yang jelas
- lisensi yang jelas
- dapat diverifikasi
- dapat diperbarui

Project Mizan tidak menggunakan data yang asal-usulnya tidak diketahui.
